# Test Configurations

This directory contains the test configurations for testing the test framework.

## Structure

1. `yaml_test_config_list.yaml` - Designed to test functionality of list type.
2. `yaml_test_config_range_list.yaml` - Designed to test functionality of range type and list type.
3. `yaml_test_config_value.yaml` - Designed to test functionality of value type.
4. `yaml_test_tolerances.yaml` - Designed to test configurable tolerance multipliers for matrix comparison.
5. `yaml_test_tolerances_invalid.yaml` - Designed to test validation of invalid tolerance values.
